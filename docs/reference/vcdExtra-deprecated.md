# Deprecated Functions in vcdExtra Package

These functions are provided for compatibility with older versions of
the vcdExtra package only. They are replaced by
[`LRstats`](https://friendly.github.io/vcdExtra/reference/LRstats.md).

## Usage

    summarise(...)
    <!-- %summarise.glm(...) -->
    <!-- %summarise.glmlist(...) -->
    <!-- %summarise.loglm(...) -->
    <!-- %summarise.loglmlist(...) -->

## Arguments

- ...:

  pass arguments down.

## Details

`summarise.*` have been replaced by
[`LRstats`](https://friendly.github.io/vcdExtra/reference/LRstats.md)
functions.
